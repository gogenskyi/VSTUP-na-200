# Vstup-na-200

Проєкт для збору та аналізу даних вступної кампанії України.

Парсер отримує дані з **abit-poisk.org.ua**, зберігає їх у базі даних SQLite та дозволяє аналізувати конкурсні пропозиції, вступників і заяви.

## Стек

* Python 3.12+
* SQLite
* Requests
* BeautifulSoup4

## Можливості

* Збір даних про університети
* Збір конкурсних пропозицій
* Збір даних про вступників
* Збереження даних у SQLite
* Аналіз конкурсних балів
* Симуляція алгоритму широкого конкурсу

## Встановлення

```bash
git clone https://github.com/USERNAME/Vstup-na-200.git
cd Vstup-na-200

python -m venv .venv
source .venv/bin/activate

pip install -r requirements.txt
```

### requirements.txt

```txt
requests
beautifulsoup4
lxml
```

## Використання

### 1. Збір даних

Запустіть парсер:

```bash
python main.py
```

Після завершення буде створена та заповнена база даних SQLite (`vstup.db` або інша, вказана у налаштуваннях проєкту).

### 2. Аналіз широкого конкурсу

Після збору даних запустіть модуль конкурсу:

```bash
python competition/competition.py
```

Скрипт використовує дані з бази та моделює розподіл бюджетних місць відповідно до пріоритетів заяв вступників і конкурсних балів.

## Застереження

Результати, отримані за допомогою `competition.py`, є лише наближеним моделюванням алгоритму широкого конкурсу та можуть відрізнятися від фактичних результатів вступної кампанії.

Причини можливих відмінностей:

* неповні або застарілі дані з відкритих джерел;
* відсутність частини службової інформації, доступної ЄДЕБО;
* спрощення окремих етапів алгоритму розподілу місць державного замовлення;
* зміни правил вступної кампанії;
* технічні помилки парсингу або обробки даних.

Проєкт не є офіційним сервісом МОН або ЄДЕБО. Усі результати слід використовувати виключно для ознайомлення, аналізу та оцінки власних шансів на вступ.
